Telegram RSS Feed Project

Welcome to the Telegram RSS Feed generator!

Available Files:

  • 📡 JSON Feed (rss_feed.json) - The generated JSON feed
  • 🐍 Python Script (rss_telegram.py) - Source code
  • 📋 Requirements (requirements.txt) - Python dependencies

Helper Scripts:

  • ⚡ Manual RSS Update (manual_rss_update.bat) - Run RSS update manually
  • 🚀 Advanced Git Push (git_push_advanced.bat) - Interactive Git push
  • 📤 Quick Git Push (git_push.bat) - Simple Git push
  • 📥 Git Pull (git_pull.bat) - Pull updates from GitHub

About this project:

This project converts Telegram channel messages into a JSON feed format.

JSON Feed URL: https://plaszeb.github.io/telegram-rss-feed/rss_feed.json

Note: Helper scripts (.bat files) can be downloaded and used locally on Windows.
